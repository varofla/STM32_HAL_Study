# STM32_HAL_Study

> 작성된 내용에는 기술적 오류가 있을 수 있으며, 블로그 댓글로 알려주시면 빠르게 수정하겠습니다!

> Windows 10 및 macOS(M1)에서 문제 없이 따라하실 수 있습니다.
